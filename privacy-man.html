Privacy Policy

For Mow Man – your cold, calculated AI-powered assistant

1. Information We Collect

Discord User and Server Information:

  • User ID, username, and display name
  • Server ID, server name, icon, channel/role lists, emoji lists

Message and Interaction Data:

  • Recent messages (100 in servers, 50 in DMs)
  • Command usage logs
  • Image URLs for Gemini API analysis

User-Provided Info: Via ^addinfo, including name, age, description

Server Audit Log Data: Last 20 actions (joins, edits, deletes, etc.)

Configuration Data: Prefix, rules, custom server settings

2. How We Use Your Information

  • To operate and execute commands
  • To provide context to Gemini AI
  • To personalize replies
  • To debug and improve features
  • To retain your server’s custom setup

3. Information Sharing

  • No selling or trading of data
  • Shared only with Gemini API (Google) for replies
  • May be disclosed if required by law

4. Data Retention

  • Message memory: overwritten or cleared on restart
  • Audit log: stores last 20 events
  • ^addinfo: stored until you delete it
  • Settings: stored until you reset/remove bot

5. Your Rights

  • You can delete data via ^deleteinfo
  • Removing the bot deletes data within 7 days

6. Policy Updates

We may update this policy. Changes are posted here with a new “Last Updated” date.

7. Contact Us

For questions, visit our Support Server.
